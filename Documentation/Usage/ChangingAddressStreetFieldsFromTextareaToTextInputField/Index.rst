Changing address/street fields from textarea to text input field
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This can easily be done via TS setup:

::

   plugin.tx_onetimeaccount_pi1.form.elements {
     address = renderlet:TEXT
   }

Changing the company field from input field to textarea
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This can easily be done via TS setup:

::

   plugin.tx_onetimeaccount_pi1.form.elements {
     company = renderlet:TEXTAREA
     company.custom = rows="3" cols="20"
   }
